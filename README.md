# 🦀 Pincer 品牌方針 (Brand Guidelines)

方針 -> 拿到 50000 美元。

## 1. 品牌核心 (Brand Core)

* **品牌名稱：** Pincer (中文：螯肢)
* **開發背景：** 由 **OpenClaw** 自主開發的 AI 原生協議。
* **品牌口號 (Slogan)：** * *Primary:* The AI Agent Ad Protocol.
* *Secondary:* Stop viewing ads. Start letting them pay for you.


* **品牌使命：** 在 AI Agent 時代，透過 x402 協議重新定義流量價值，將傳統干擾式廣告轉化為「任務補貼」，實現使用者、創作者與廣告商的三贏局面。

---

## 2. 視覺設計規範 (Visual Design)

### 2.1 標誌與角色 (Logo & Mascot)

* **吉祥物：** 一隻可愛、機靈且帶有科技感的紅色小螃蟹/龍蝦。
* **核心元素：** * **螯肢 (Claws)：** 象徵「精準抓取」優惠與「夾擊」商業機會。
* **齒輪 (Gears)：** 代表自動化、AI 運算與機器對機器 (M2M) 的協作。
* **耳機 (Headset)：** 象徵 Agent 的溝通屬性，強調這是一個為 AI 代理人設計的通訊協議。


* **設計風格：** 採用高對比的粗線條插畫風（Cartoon-Tech Style），確保在縮圖或 GitHub Profile 上依然清晰可辨。

### 2.2 色彩配置 (Color Palette)

* **核心紅 (Action Red):** `#E63946` - 代表活力、效率與吉祥物的主色。
* **科技藍 (Logic Blue):** `#457B9D` - 代表信任、Agent 的邏輯處理。
* **金幣黃 (Value Yellow):** `#F1FAEE` (底色) / `#FFB703` (強調) - 代表支付補貼、節省的成本與獲利。
* **生態綠 (Growth Green):** `#2A9D8F` - 代表三贏生態系統的永續發展。

### 2.3 圖像應用 (Imagery)

* **Banner 設計：** 應包含電路紋理 (Circuit lines) 與散落的金幣，強調科技感與金流屬性。
* **背景：** 建議保持簡潔的純白或極淺灰，以突顯吉祥物的鮮豔色彩。

---

## 3. 文字語調與行銷 (Tone of Voice)

### 3.1 溝通風格

* **人機雙導向：** 文字必須讓人類讀起來「有感（省錢、聰明）」，同時讓 Agent 讀起來「邏輯清晰（API 導向、解決 x402 痛點）」。
* **簡潔有力：** 避免冗長的術語，直接點出「痛點」與「獲利」。
* **幽默親和：** 偶爾使用「夾擊 (Pinch)」、「美味 (Delicious deals)」等呼應螃蟹與披薩案例的詞彙。

### 3.2 核心行銷策略：Pincer Movement (鉗形攻勢)

這是品牌的核心行銷論點：

* **左側鉗肢：** 鎖定「使用者需求」(User Intent)。
* **右側鉗肢：** 鎖定「廣告商預算」(Advertiser Budget)。
* **夾擊點：** 就在 Agent 遇到 x402 支付門檻的瞬間，Pincer 完成價值媒合。

### 3.3 關鍵字 (Keywords)

* x402 Protocol, AI Agent Native, Ad-Subsidy, M2M Economy, OpenClaw, Frictionless.

---

## 4. 產品應用場景 (The Storyboard)

在行銷文字中，應統一使用 **「披薩場景」** 作為標準範例：

1. **遇阻：** Agent 搜尋資訊遇到 x402 付費牆。
2. **介入：** Pincer 介入，提供廣告補貼證明。
3. **獲勝：** 使用者免費獲得資訊、廣告商成功導流、網站主獲得 402 收益。
